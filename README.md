A **MERN full-stack** web application for listing and browsing properties (houses) in Nepal. Users can **view, search, upload properties**, see **images and locations on the map**, and interact with the platform easily.

## Features
- User authentication (Login/Register)
- Upload properties with multiple images
- Property details: price, rooms, bathrooms, size, distance from main road
- Interactive map showing all property locations
- Responsive UI for mobile & desktop
- RESTful API backend with MongoDB

## Tech Stack
- **Frontend:** React, TailwindCSS, Leaflet.js
- **Backend:** Node.js, Express.js, MongoDB, Mongoose
- **Authentication:** JWT
